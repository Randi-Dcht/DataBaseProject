Menu

Le menu de cette application comporte 4 choix :
- utiliser un dataBase qui existe déjà.
- créer et utiliser une nouvelle dataBase.
- voir la documentation de cette application
- quitter totalement cette application

Utiliser dataBase

Un choix déroulant, vous propose plusieurs choix de dataBase existant dans l'application.
Si ce choix est vide, c'est que vous n'avez pas encore sauveguader une dataBase avec cette application.

Utiliser une nouvelle dataBase

Dans le menu princiaple de l'application, vous pouvez entrer le nom de la nouvelle dataBase.
Puis cliquer sur "ok" pour aller vers la création de la dataBase.

Création dataBase

Vous arrivez sur une fenêtre où vous devez choisir le nom de votre première table.
Ensuite, vous pouvez ajouter des attributs à cette table via le bouton "add attributes" en haut à gauche.
Pour valider le nouvel attribut, cliquer sur le bouton "OK" qui se trouve à droite du champs.

Pour ajouter une nouvelle table dans cette dataBase, vous pouvez cliquer sur "add table" qui se trouve en bas de la fenêtre.
Donner lui un nom puis vous pouvez ajouter des attributs à celle-ci.

Utilisation dataBase

Dependence

Dans cette partie de l'application, vous pouvez :
- ajouter des dépendances à la base de donnée.
- supprimer des dépendances à la base de donnée.

DataBase

Dans cette partie de l'application, vous pouvez :
- voir les tables qui se trouve dans la base de donnée.
- ajouter de nouvelles tables avec "add table".
- ajouter de nouveaux attributs dans les nouvelles tables avec "add attribute".
/!\ /!\ quand vous avez finie, vous pouvez cliquer sur "valid". /!\ /!\

Option

Dans cette partie de l'application, vous pouvez :
- vérifier si la base est en 3NF.
- vérifier si la base est en BCNF.
- voir toutes les clées de la base de donnée.
