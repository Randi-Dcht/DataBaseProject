This project is a data base with the dependence.
In this project, we use the sqlLite and language Java. Funthemore, we use JavaFx for graphics and Junit 4 for test.

The commands of this project are:
 * ant build for compile all project.
 * ant run for launch all application (graphic and line command).
 * ant clean for clean the repertory.
 * ant cmd for launch only line command application without graphic.
 * ant gui for launch only graphic application.
 * ant test for launch the test of Junit.

 /!\ If a problem with Javafx, launch with ant cmd and ant test only! /!\
